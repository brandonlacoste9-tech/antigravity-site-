# Canada Daily

A modern, high-performance news and analysis platform for Canada.

## Features
- **SSR (Server-Side Rendering)**: Powered by TanStack Start and Nitro.
- **Dynamic Feed**: Real-time news via CBC RSS.
- **Bi-lingual**: English (primary) and French support.
- **Modern UI**: Clean, editorial design with Dark Mode support.

## Tech Stack
- TanStack Start (SSR)
- Vite
- Tailwind CSS 4
- Lucide Icons
- RSS Parser

## Development
```bash
pnpm install
pnpm dev
```

## Deployment
Optimized for Vercel.
